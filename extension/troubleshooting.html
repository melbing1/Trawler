Troubleshooting

An issue has occurred that prevented the Trawler extension from working properly.
Please verify the following:

  1. You are connected to the internet
  2. You have not blocked access to Trawler server (this can also cause performance issues in some cases)
  3. You have given the extension the permission that it requested
  4. You are using an up-to-date version of Firefox
